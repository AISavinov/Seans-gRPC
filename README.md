# Seans-gRPC

My gRPC experiments

## setup
```bash
git clone https://github.com/Sean-Bradley/Seans-gRPC.git
$ virtualenv venv
$ source venv/Scripts/activate
$ pip install -r requirements.txt
```

## Start Server
```bash
python -m server
```

## Start Client
```bash
python -m client
```

## Start Multiple clients in sperate processes
```bash
python -m multiprocess
```
